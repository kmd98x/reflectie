# Omnient Reflect - Drowning Person Tool

A mobile web application for self-reflection and leadership development, helping users discover their behavioral patterns through a series of statements.

## Tech Stack

- React with JavaScript
- Tailwind CSS for styling
- React Router for navigation
- Vite for build tooling

## Features

- 20 randomized statements across 5 behavioral patterns
- Pattern detection based on user responses
- Personalized coaching questions
- Appointment booking flow
- Mobile-first design (375px width)

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Behavioral Patterns

1. **TRY HARD** - Taking on too much, solving everything alone
2. **PLEASE** - Difficulty saying no, avoiding conflict
3. **BE PERFECT** - High self-expectations, fear of mistakes
4. **BE STRONG** - Hiding vulnerability, not asking for help
5. **HURRY UP** - Always rushing, difficulty slowing down

## Design

- Primary Color: #2C3E50 (deep navy)
- Accent Color: #E8A87C (warm peach)
- Background: #FFFFFF (white)
- Text: #333333 (dark gray)
- Typography: Inter sans-serif

## Screen Flow

1. Zero State (Welcome)
2. Name Input
3. Privacy Policy
4. Current Situation
5. Homepage/Instructions
6. 20 Statement Screens (randomized)
7. Pattern Detection
8. Pattern Result
9. Coaching Question
10. Appointment Booking
11. Success Screen
